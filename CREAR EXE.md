# Crear el archivo .exe de la aplicación

## Código para crear el ejecutable

```
 pyinstaller --onefile main.py
 ^^^^^^^^^^^ ^^^^^^^^^ ^^^^^^^
      │          │       └─ Archivo principal de la aplicación
      │          └─ Crea un unico archivo .exe
      └─ Herramienta para converitr un archivo .py en cualquier ejecutable 

```

Después de realizar los cambios y una vez confirmada la nueva versión de la aplicación se crea .exe como único archivo, este está listo para compartir con los usuarios.
